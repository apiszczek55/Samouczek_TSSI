Czym jest HTML?

HTML (Hypertext Markup Language) to język znaczników używany do tworzenia stron internetowych. Do definiowania różnych elementów strony używamy znaczników. Organizacją odpowiedzialną za rozwój standardów internetowych jest W3C (World Wide Web Consortium). Najnowszą wersją HTML jest obecnie HTML 5.

Podstawowa struktura dokumentu HTML:

  1. <!DOCTYPE html> - deklaracja typu dokumentu (język HTML5)
  2. <html> - element root (posiada atrybut lang określający język dokumentu)
  3. <head> - sekcja nagłówkowa. Zawiera metadane, tytuł oraz informacje niewidoczne bezpośrednio na stronie
  4. <meta charset="UTF-8"> - deklaracja strony kodowej
  5. <title> - tytuł strony wyświetlany na karcie przeglądarki
  6. <body> - treść strony
  7. <!--komentarz--> - komentarz
<!DOCTYPE html>
<html lang="pl">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, 
initial-scale=1.0">
    <title>Moja strona</title>
</head>
<body>
    <h1>Witaj na mojej stronie!</h1>
    <p>To jest prosty paragraf.</p>
</body>
</html>

Czym jest walidator?

Walidator to narzędzie, które służy do sprawdzania poprawności kodu HTML. Pozwala ono na wykrycie błędów w kodzie i zapewnia, że strona jest zgodna z obowiązującymi standardami.

Powiązane pojęcia

  1. JavaScript - język programowania, który pozwala na tworzenie interaktywnych elementów na stronach internetowych.
  2. SEO (Search Engine Optimization) - praktyki mające na celu poprawę widoczności strony w wynikach wyszukiwania.
  3. Dostępność cyfrowa - projektowanie witryn użytecznych dla wszystkich, w tym osób z niepełnosprawnościami, zgodnie ze standardem WCAG 2.1.
  4. Strona internetowa - dokument HTML udostępniany w Internecie.
  5. Przeglądarka - program komputerowy służący do wyświetlania stron internetowych. Jej zadaniami są kreowanie zawartości strony, wysyłanie żądań i pobieranie stron.
  6. Portal - strona internetowa oferująca dostęp do szerokiego zakresu informacji, która zawiera linki do innych stron i usług.
  7. Wortal - strona internetowa skupiająca się na jednej tematyce.
