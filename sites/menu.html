Menu

Menu najlepiej umieszczać na stronie w elemencie nav.

Może mieć ono formę:

  • pionową
  • poziomą

Należy pamiętać o użyciu listy nieuporządkowanej (ul) i usunięciu jej punktacji.

Menu pionowe

Aby stworzyć pionowe menu można:

  • ustawić elementy listy na display: block
  • usunąć dekorację hiperłącza
ul{
    list-style-type: none;
}
ul li{
    display: block;
}
ul a{
    display: block;
    width: 100px;
    text-decoration: none;
    color: black;
}

Menu poziome

Aby stworzyć poziome menu można:

  • ustawić elementy listy na display: inline-block
  • usunąć dekorację hiperłącza
ul{
    list-style-type: none;
}
ul li{
    display: inline-block;
}
ul a{
    display: block;
    width: 100px;
    text-decoration: none;
    color: black;
}
