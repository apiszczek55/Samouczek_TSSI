Formatowanie list

list-style-type - określa typ punktacji listy (marker listy)

Najczęstsze wartości:

  • circle - okręgi
  • disc - kropki (domyślna wartość dla list nieuporządkowanych)
  • decimal - liczby (domyślna wartość dla list uporządkowanych)
  • decimal-leading-zero - liczby z zerami na początku
  • lower-alpha - małe litery alfabetu
  • square - kwadraty
  • none - brak punktacji

Inne wartości:

  • armenian
  • cjk-ideographic
  • georgian
  • hebrew
  • hiragana
  • hiragana-iroha
  • katakana
  • katakana-iroha
  • lower-greek - małe litery greckie
  • lower-latin - małe litery łacińskie
  • lower-roman - małe cyfry rzymskie
  • upper-alpha - wielkie litery alfabetu
  • upper-greek - wielkie litery greckie
  • upper-latin - wielkie litery łacińskie
  • upper-roman - wielkie cyfry rzymskie
ul{
        list-style-type: square;
    }
    ol{
        list-style-type: upper-greek;
    }

list-style-image - określa obraz jako punktację listy

ul{
        list-style-image: url('kwiatek.png');
    }

list-style-position - określa położenie punktacji listy (markera listy)

  • inside - marker znajduje się wewnątrz elementu listy
  • outside - marker znajduje się na zewnątrz elementu listy (domyślna wartość)
ul{
        list-style-position: inside;
    }
    ol{
        list-style-position: outside;
    }

list-style - skrócona właściwość dla list-style-type, list-style-position i list-style-image

ul{
        list-style: square inside;
    }
    ol{
        list-style: upper-greek outside;
    }
