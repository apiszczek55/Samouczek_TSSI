Właściwości CSS

W CSS istnieje wiele właściwości, które pozwalają na kontrolowanie wyglądu i układu elementów na stronie. Poniżej znajduje się kilka podstawowych właściwości, które są często używane.

Tekst

color - kolor tekstu

p{
    color: red;
}

font-family - rodzaj czcionki. Wyrózniamy kilka rodzajów czcionek:

  • serif - czcionki z ozdobnikami na końcach (np. "Times New Roman", Georgia). Zawyczaj używane do tekstu drukowanego
  • sans-serif - czcionki bez ozdobników na końcach (np. Arial, Helvetica). Zawyczaj używane do tekstu na urządzeniach cyfrowych
  • monospace - czcionki o stałej szerokości (np. "Courier New", Monaco). Zawyczaj używane do kodu źródłowego
  • cursive - czcionki przypominające pismo odruckowe (np. "Brush Script"). Są używane do dekoracji
  • fantasy - czcionki dekoracyjne (np. Impact, "Comic Sans MS"). Często używane do nagłówków i tytułów

Warto pamiętać, aby określić kilka czcionek jako alternatywy, jeśli jedna z nich nie jest dostępna, zastosowana zostanie kolejna. Na końcu należy określić główną czcionkę, z której podane czcionki pochodzą (serif, sans-serif, monospace, cursive, fantasy). Nazwy czcionek, które składają się z wielu wyrazów, należy zapisywać w cudzysłowie.

p{
    font-family: Arial, sans-serif;
}
h2{
    font-family: "Times New Roman", serif;
}

font-size - rozmiar czcionki

p{
    font-size: 16px;
}

font-weight - grubość czcionki

  • normal - normalna grubość (400)
  • bold - pogrubiona (700)
  • bolder - grubsza od odziedziczonej lub przypisanej do znacznika (wartość względna)
  • lighter - cieńsza od odziedziczonej lub przypisanej do znacznika (wartość względna)
  • 100-900 - wartości numeryczne, gdzie 100 jest najcieńsza, a 900 jest najgrubsza
p{
    font-weight: bold;
}

font-style - styl czcionki

  • normal - normalny styl
  • italic - kursywa
  • oblique - bardziej pochyły od kursywy
p{
    font-style: italic;
}

text-decoration-line - dekoracja tekstu

  • none - brak dekoracji
  • underline - podkreślenie
  • overline - linia nad tekstem
  • line-through - przekreślenie

text-decoration-color - kolor dekoracji tekstu

text-decoration-style - styl dekoracji tekstu

  • solid - ciągła linia
  • dashed - przerywana linia
  • dotted - kropkowana linia
  • double - podwójna linia
  • wavy - falista linia

text-decoration-thickness - grubość dekoracji tekstu

  • auto
  • from-font
  • jednostka

text-decoration - skrót dla wszystkich właściwości dekoracji tekstu

p{
    text-decoration-line: underline;
    text-decoration-color: red;
    text-decoration-style: solid;
    text-decoration-thickness: auto;
}
a{
    text-decoration: underline blue dashed 2px;
}

text-align - wyrównanie tekstu

  • left - wyrównanie do lewej
  • right - wyrównanie do prawej
  • center - wyrównanie do środka
  • justify - wyrównanie na obie strony
p{
    text-align: center;
}

font-variant - wariant czcionki. small-caps wyświetla tekst w małych kapitalikach (wielkie litery są wyświetlane w mniejszym rozmiarze, a małe litery są zamieniane na wielkie w mniejszym rozmiarze).

p{
    font-variant: small-caps;
}

text-shadow - cień tekstu

p{
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
}

text-transform - transformacja tekstu

  • none - brak transformacji
  • capitalize - pierwsza litera każdego słowa jest zamieniana na wielką
  • uppercase - wszystkie litery są zamieniane na wielkie
  • lowercase - wszystkie litery są zamieniane na małe
p{
    text-transform: uppercase;
}

line-height - wysokość linii (odstęp między liniami tekstu)

p{
    line-height: 1.5;
}

letter-spacing - odstęp między literami

p{
    letter-spacing: 2px;
}

word-spacing - odstęp między słowami

p{
    word-spacing: 2px;
}

text-indent - wcięcie pierwszej linii tekstu

p{
    text-indent: 20px;
}

white-space - kontroluje, jak przeglądarka obsługuje białe znaki (spacje, tabulatory, nowe linie)

  • normal - domyślne zachowanie, białe znaki są redukowane do pojedynczych spacji, a tekst jest zawijany
  • nowrap - tekst nie jest zawijany
  • pre - białe znaki są zachowywane, a tekst nie jest zawijany (podobnie jak w elemencie <pre>). Tekst jest wyświetlany z zachowaniem oryginalnego formatowania jak w kodzie źródłowym
  • pre-wrap - białe znaki są zachowywane, a tekst jest zawijany
  • pre-line - białe znaki są redukowane do pojedynczych spacji, a tekst jest zawijany tylko na nowych liniach
p{
    white-space: pre-wrap;
}

Wysokość i szerokość

width - szerokość elementu

height - wysokość elementu

div{
    width: 200px;
    height: 100px;
}

min-width - minimalna szerokość elementu

max-width - maksymalna szerokość elementu

div{
    min-width: 100px;
    max-width: 200px;
}

min-height - minimalna wysokość elementu

max-height - maksymalna wysokość elementu

div{
    min-height: 50px;
    max-height: 150px;
}

Tło

background-color - kolor tła

    • nazwy kolorów - np. red, blue, green
    • RGB - rgb(255, 0, 0) dla czerwonego
    • RGBA - rgba(255, 0, 0, 0.5) dla półprzezroczystego czerwonego
    • HSL - hsl(0, 100%, 50%) dla czerwonego
    • HSLA - hsla(0, 100%, 50%, 0.5) dla półprzezroczystego czerwonego
    • heksadecymalne - #ff0000 dla czerwonego
    • heksadecymalne z przezroczystością - #ff000080 dla półprzezroczystego czerwonego
    • transparent - przezroczyste tło (domyślne)
p{
    background-color: lightgray;
}

background-image - obraz tła lub gradient

body{
    background-image: url('obraz.jpg');
}

background-repeat - powtarzanie obrazu tła

  • repeat - obraz jest powtarzany zarówno w poziomie, jak i w pionie (domyślne)
  • repeat-x - obraz jest powtarzany tylko w poziomie
  • repeat-y - obraz jest powtarzany tylko w pionie
  • no-repeat - obraz nie jest powtarzany
body{
    background-repeat: no-repeat;
}

background-size - rozmiar obrazu tła

  • auto - domyślny rozmiar obrazu
  • cover - obraz jest skalowany, aby całkowicie pokryć tło, zachowując proporcje (może być obcięty)
  • contain - obraz jest skalowany, aby całkowicie zmieścić się w tle, zachowując proporcje (może pozostać puste tło)
  • jednostka - można określić rozmiar obrazu za pomocą jednostek (np. 100px 50px)
body{
    background-size: cover;
}

background-position - pozycja obrazu tła

  • left
  • center
  • right
  • top
  • bottom
  • jednostka
body{
    background-position: center;
}

background-attachment - kontroluje, czy tło przewija się wraz z zawartością strony

  • scroll - tło przewija się wraz z zawartością (domyślne)
  • fixed - tło jest przyklejone do okna przeglądarki i nie przewija się
body{
    background-attachment: fixed;
}

background - skrót dla wszystkich właściwości tła

body{
    background: lightgray url('obraz.jpg') no-repeat center;
}

Overflow

overflow - kontroluje, jak zawartość elementu jest wyświetlana, gdy przekracza jego granice (w pionie i poziomie)

  • visible - zawartość jest wyświetlana poza granicami elementu (domyślne)
  • hidden - zawartość jest ukrywana poza granicami elementu
  • scroll - dodawane są paski przewijania, aby móc zobaczyć ukrytą część zawartości
  • auto - przeglądarka decyduje, czy dodać paski przewijania
div{
    overflow: hidden;
}

overflow-x - kontroluje przewijanie w poziomie

overflow-y - kontroluje przewijanie w pionie

div{
    overflow-x: hidden;
    overflow-y: hidden;
}
