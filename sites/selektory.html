Selektory elementów

* - selektor ogólny, wybiera wszystkie elementy na stronie. Jest często używany do zerowania domyślnych dla przegladarki marginesów i paddingów (o których dowiesz się w części "Model pudełkowy")

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

element - selektor elementu, wybiera wszystkie elementy danego typu

p {
    color: red;
}

.klasa - selektor klasy, wybiera wszystkie elementy z daną klasą

.zaznaczenie {
    background-color: yellow;
}

#id - selektor identyfikatora, wybiera element o danym identyfikatorze

#kontener {
    background-color: lightgray;
}

element, element2 - selektor grupowy, wybiera wszystkie elementy wymienione w grupie

h1, h2 {
    font-family: Arial, sans-serif;
}

element element2 - selektor potomka, wybiera wszystkie elementy2 będące potomkami danego elementu (są gdzieś w jego środku)

nav ul {
    list-style: none;
}

element > element2 - selektor dziecka, wybiera wszystkie elementy2 będące bezpośrednimi dziećmi danego elementu (są bezpośrednio w jego wnętrzu)

nav > ul {
display: flex;
}

element + element2 - selektor braci, wybiera element2 będący bezpośrednim następnikiem danego elementu

h1 + p {
    color: blue;
}

element ~ element2 - ogólny selektor braci, wybiera wszystkie elementy2 będące po danym elemencie

h1 ~ p {
    color: green;
}

Selektory pseudoklas i pseudoelementów

element:pseudoklasa - selektor pseudoklasy, wybiera elementy w określonym stanie

a:hover {        <--link po najechaniu myszką-->
    color: red;
}
button:active {        <--button podczas naciśnięcia-->
    color: blue;
}
a:visited {        <--link odwiedzony-->
    color: purple;
}
li:nth-child(odd) {        <--nieparzyste elementy listy-->
    background-color: lightgray;
}

element::pseudoelement - selektor pseudoelementu, wybiera część elementu

p::first-letter {        <--pierwsza litera akapitu-->
    font-size: 2em;
    font-weight: bold;
}
p::first-line {        <--pierwsza linia akapitu-->
    font-weight: bold;
}
li::marker {        <--marker listy-->
    color: red;
}

Selektory atrybutów

[atrybut] - selektor atrybutu, wybiera wszystkie elementy posiadające dany atrybut

[title] {
    border: 1px solid blue;
}

[atrybut="wartość"] - selektor, który wybiera wszystkie elementy posiadające dany atrybut o konkretnej wartości

[title="ważne"] {
    color: green;
}

[atrybut~="wartość"] - selektor, który wybiera wszystkie elementy posiadające dany atrybut, którego wartość jest jedną z wielu wartości oddzielonych spacją

[class~="zaznaczenie"] {
    background-color: yellow;
}

[atrybut^="wartość"] - selektor, który wybiera wszystkie elementy posiadające dany atrybut, którego wartość zaczyna się od podanej wartości

[href^="https"] {       <--linki zaczynające się od https-->
    color: green;
}

[atrybut$="wartość"] - selektor, który wybiera wszystkie elementy posiadające dany atrybut, którego wartość kończy się na podaną wartość

[href$=".pdf"] {       <--pliki kończące się na .pdf-->
    color: red;
}

[atrybut*="wartość"] - selektor, który wybiera wszystkie elementy posiadające dany atrybut, którego wartość zawiera podaną wartość

[href*="example"] {     <--elementy zawierające w href "example"-->
    color: blue;
}

Zasady nazewnictwa klas i identyfikatorów

  1. Nie używaj spacji. Używaj -, _ lub wielkich liter do oddzielania słów
  2. Wielkość liter ma znaczenie
  3. Twórz opisowe nazwy
  4. Nie używaj cyfr na początku
