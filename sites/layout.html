Ważne dla układu stron znaczniki

<div> - element blokowy, służy do grupowania innych elementów i tworzenia sekcji na stronie

Znaczniki semantyczne - określają swoją zawartość i znaczenie w kontekście dokumentu

  • <header> - reprezentuje nagłówek strony lub sekcji
  • <nav> - sekcja nawigacyjna z linkami do innych stron lub sekcji
  • <main> - główna treść dokumentu
  • <footer> - stopka strony lub sekcji
  • <section> - tematyczna grupa treści, zazwyczaj z nagłówkiem
  • <article> - samodzielna treść, taka jak wpis na blogu lub artykuł
  • <aside> - treść poboczna, taka jak pasek boczny lub reklama (powinna być pośrednio powiązana z otaczającą ją treścią - ilustrować, uzupełniać)
  • <figure> - element do grupowania obrazów i ich opisów
  • <figcaption> - opis dla elementu <figure>

Właściwość display

display - ważna właściwość CSS służąca do kontrolowania układu. Określa jak element HTML jest traktowany

Wartości display:

  • block - element jest traktowany jako blok, zajmuje całą szerokość dostępnego miejsca i zaczyna się od nowej linii
  • inline - element jest traktowany jako liniowy, zajmuje tylko tyle miejsca, ile potrzebuje, i nie zaczyna się od nowej linii
  • inline-block - element jest traktowany jako liniowy, ale można mu ustawić szerokość i wysokość
  • none - element jest ukryty i nie zajmuje żadnego miejsca na stronie
  • flex - element jest traktowany jako kontener flex. Flexbox to model układu umożliwiający elastyczne i responsywne rozmieszczanie elementów (poziomo lub pionowo) w kontenerze
  • grid - element jest traktowany jako kontener grid i umożliwia stworzenie układów opartych na siatce z wierszami i kolumnami

Float i clear

Właściwość float - służy do przemieszczania elementów wokół innych elementów. Jest używana głównie do tworzenia układów z elementami przypinanymi do lewej lub prawej krawędzi kontenera, a także do otaczania obrazów tekstem.

Wartości float:

  • left - element unosi się na lewo od kontenera
  • right - element unosi się na prawo od kontenera
  • none - element nie jest pływający i jest wyświetlany w miejscu, w którym występuje w tekście

Właściwość clear - służy do kontrolowania, jak elementy reagują na elementy pływające (float). Określa, czy element powinien być umieszczony obok elementów pływających, czy poniżej nich.

Wartości clear:

  • none - pozwala elementom swobodnie poruszać się po obu stronach (domyślny)
  • left - element jest przesuwany poniżej elementów pływających po jego lewej stronie
  • right - element jest przesuwany poniżej elementów pływających po jego prawej stronie
  • both - element jest przesuwany poniżej elementów pływających

Przykład:

nav {
    float: left;
}
main{
    float: left;
}
footer{
    clear: both;
}

Flexbox

Flexbox to model układu w CSS, który umożliwia elastyczne i responsywne rozmieszczanie elementów (poziomo lub pionowo) w kontenerze. Wymaga rodzica (kontenera) z ustawioną właściwością display: flex lub display: inline-flex oraz dzieci (elementów w kontenerze, 1 lub więcej).

Właściwości związane z Flexboxem:

  1. display: flex - określa, że element jest kontenerem flex
  2. display: inline-flex - określa, że element jest liniowym kontenerem flex
  3. flex-direction - określa kierunek, w którym elementy flexbox będą rozmieszczane w kontenerze
    • row - elementy są rozmieszczane poziomo (domyślnie)
    • row-reverse - elementy są rozmieszczane poziomo, ale w odwrotnej kolejności
    • column - elementy są rozmieszczane pionowo
    • column-reverse - elementy są rozmieszczane pionowo, ale w odwrotnej kolejności
  4. flex-wrap - określa, czy elementy mogą się zawijać
    • nowrap - elementy nie zawijają się (domyślnie)
    • wrap - elementy zawijają się
    • wrap-reverse - elementy zawijają się w odwrotnej kolejności
  5. flex-flow - skrót dla flex-direction i flex-wrap
  6. justify-content - wyrównuje elementy , gdy nie wykorzystują całej dostępnej przestrzeni poziomo
    • flex-start - elementy są wyrównane do początku kontenera (domyślnie)
    • flex-end - elementy są wyrównane do końca kontenera
    • center - elementy są wyśrodkowane w kontenerze
    • space-between - elementy są rozmieszczone równomiernie, z pierwszym elementem przy początku kontenera i ostatnim przy końcu
    • space-around - elementy są rozmieszczone ze spacją wokół każdego elementu
    • space-evenly - elementy są rozmieszczone równomiernie, z równą ilością miejsca wokół każdego elementu
  7. align-items- wyrównuje elementy , gdy nie wykorzystują całej dostępnej przestrzeni pionowo (wartości normal i stretch są tym samym)
    • normal - elementy rozciągają się, aby wypełnić kontener (domyślnie)
    • flex-start - elementy są wyrównane do początku kontenera
    • flex-end - elementy są wyrównane do końca kontenera
    • center - elementy są wyśrodkowane w kontenerze
    • stretch - elementy rozciągają się, aby wypełnić kontener (domyślnie)
    • baseline - elementy są wyrównane względem linii bazowej ich treści
  8. align-content - wyrównuje linie elementów, gdy nie wykorzystują całej dostępnej przestrzeni pionowo
    • flex-start - linie są wyrównane do początku kontenera
    • flex-end - linie są wyrównane do końca kontenera
    • center - linie są wyśrodkowane w kontenerze
    • space-between - linie są rozmieszczone równomiernie, z pierwszą linią przy początku kontenera i ostatnią przy końcu
    • space-around - linie są rozmieszczone ze spacją wokół każdej linii
    • space-evenly - linie są rozmieszczone równomiernie, z równą ilością miejsca wokół każdej linii

Elementy flexbox mogą mieć takie właściowości jak:

  1. order - określa kolejność elementów w kontenerze flex
  2. flex-grow - określa, jak bardzo element może rosnąć w stosunku do innych elementów
  3. flex-shrink - określa, jak bardzo element może się kurczyć w stosunku do innych elementów
  4. flex-basis - określa początkowy rozmiar elementu przed zastosowaniem właściwości flex-grow i flex-shrink
  5. flex - skrót dla flex-grow, flex-shrink, i flex-basis
  6. align-self - pozwala na nadpisanie ustawienia wyrównania dla pojedynczego elementu w kontenerze flex

Centrowanie kontenera:

.kontener{
    display: flex;
    justify-content: center;
    align-items: center;
}

Grid layout

Grid to system układu oparty na siatce z wierszami i kolumnami. Wymaga rodzica (kontenera) z ustawioną właściwością display: grid lub display: inline-grid oraz dzieci (elementów w kontenerze, 1 lub więcej).

Właściwości związane z Grid layout:

  1. display: grid - określa, że element jest kontenerem grid
  2. display: inline-grid - określa, że element jest liniowym kontenerem grid
  3. grid-template-columns - definiuje kolumny w siatce. Może mieć wartości takie jak:
    • px
    • fr
    • %
    • auto
    • repeat() - powtarza określoną liczbę razy określoną wartość (np. repeat(3, 1fr))
    • minmax() - określa zakres wielkości
      (np. minmax(100px, 1fr))
  4. grid-template-rows - definiuje wiersze w siatce. Może mieć wartości takie jak:
    • px
    • fr
    • %
    • auto
    • repeat()
    • minmax()
    • min-content
    • max-content
    • fit-content
  5. grid-template-areas - definiuje sposób wyświetlania kolumn i wierszy przy użyciu nazwanych elementów siatki
  6. column-gap - określa odstęp między kolumnami
  7. row-gap - określa odstęp między wierszami
  8. gap - określa odstęp między wierszami i kolumnami
  9. align-content - określa, jak elementy w siatce mają być rozłożone w pionie
    • space-evenly
    • space-around
    • space-between
    • center
    • start
    • end
  10. justify-content - określa, jak elementy w siatce mają być rozłożone w poziomie
    • space-evenly
    • space-around
    • space-between
    • center
    • start
    • end
  11. place-content - określa, jak elementy w siatce mają być rozłożone w poziomie i pionie

Elementy siatki mogą mieć właściwości:

  1. grid-column-start - określa, w którym wierszu siatki ma się zaczynać element
  2. grid-column-end - określa, w którym wierszu siatki ma się kończyć element
  3. grid-column - skrót do ustawienia zarówno grid-column-start, jak i grid-column-end
  4. grid-row-start - określa, w którym wierszu siatki ma się zaczynać element
  5. grid-row-end - określa, w którym wierszu siatki ma się kończyć element
  6. grid-row - skrót do ustawienia zarówno grid-row-start, jak i grid-row-end
  7. grid-area - skrót do ustawienia grid-row-start, grid-column-start, grid-row-end i grid-column-end
  8. grid-area - określa też nazwę elementu w siatce (do zastosowania w grid-template-areas)
  9. align-self - określa, jak element ma być wyrównany w pionie w ramach siatki
  10. justify-self - określa, jak element ma być wyrównany w poziomie w ramach siatki
  11. place-self - skrót do ustawienia zarówno align-self, jak i justify-self
  12. order
