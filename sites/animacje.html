Animacje CSS

Animacje CSS pozwalają na tworzenie efektów animacji bez użycia JavaScript. Można animować różne właściwości CSS, takie jak kolor, pozycja, rozmiar i wiele innych. Animacje są definiowane za pomocą reguły @keyframes i są dołączone do elementów za pomocą włąściwości animation.

@keyframes

Reguła @keyframes pozwala na zdefiniowanie animacji - określa się jej różne etapy i przypisuje im wartości właściwości CSS. Nazwa animacji może być zawarta w nawiasach lub bez nich. Jest to sama animacja, którą można zastosować w wielu elementach.

@keyframes nazwa-animacji {
    0% {
        transform: translate(20px, 30px);
        background-color: red;
    }
    50% {
        transform: rotate(45deg);
        background-color: blue;
    }
    100% {
        transform: scale(1.5);
        background-color: yellow;
    }
}
@keyframes nazwa-animacji2 {
    from {
        transform: translate(20px, 30px);
        background-color: red;
    }
    to {
        transform: scale(1.5);
        background-color: yellow;
    }
}

Transformacje i przejścia

Transform - właściwość umieszczana w @keyframes lub w danym elemencie. Powoduje, że element zmienia swoją pozycję, rozmiar lub kierunek. Przykładowe wartości:

  • translateX(x) - przesuwa element w poziomie
  • translateY(y) - przesuwa element w pionie
  • translate(x, y) - przesuwa element w poziomie i pionie
  • rotate(kąt) - obraca element o dany kąt
  • scaleX(x) - skaluje element w poziomie
  • scaleY(y) - skaluje element w pionie
  • scale(x, y) - skaluje element w poziomie i pionie
  • skewX(kątX) - przekrzywia element w poziomie
  • skewY(kątY) - przekrzywia element w pionie
  • skew(kątX, kątY) - przekrzywia element w poziomie i pionie
.element {
    transform: translateX(10px);
    transform: rotate(45deg);
    transform: scale(1.5);
}

Transition - właściwość przejścia. Można użyć jej m.in. do określenia czasu trwania przejścia.

  1. transition-property - określa nazwę właściwości CSS, dla której jest efekt przejścia
    • all - przejście dotyczy wszystkich właściwości CSS
    • nazwa-właściwości - przejście dotyczy tylko tej właściwości CSS (np. background-color, width)
    • none - brak przejścia
  2. transition-duration - określa czas trwania przejścia
  3. transition-timing-function - określa krzywą prędkości przejścia
    • linear - animacja przebiega równomiernie
    • ease - animacja zaczyna się powoli, przyspiesza, a następnie zwalnia
    • ease-in - animacja zaczyna się powoli i przyspiesza
    • ease-out - animacja zaczyna się szybko i zwalnia
    • ease-in-out - animacja zwalnia na początku i na końcu
    • cubic-bezier(n1, n2, n3, n4) - animacja przebiega zgodnie z krzywą określoną przez cztery wartości 0-1 (np. cubic-bezier(0.25, 0.1, 0.25, 1))
  4. transition-delay - określa opóźnienie rozpoczęcia przejścia
  5. transition - skrót dla wszystkich właściwości przejścia
div {
    width: 100px;
    transition: width 2s;
}
div:hover {
    width: 300px;
}

animation

Właściwość animation pozwala na zastosowanie animacji do elementu.

  1. animation-name - określa nazwę animacji zdefiniowanej w @keyframes
  2. animation-duration - określa czas trwania animacji
  3. animation-timing-function - określa krzywą prędkości animacji
    • linear
    • ease
    • ease-in
    • ease-out
    • ease-in-out
    • cubic-bezier(n1, n2, n3, n4)
  4. animation-delay - określa opóźnienie rozpoczęcia animacji
  5. animation-iteration-count - określa liczbę powtórzeń animacji
    • numer
    • infinite - animacja powtarza się w nieskończoność
  6. animation-direction - określa kierunek animacji
    • normal - animacja odtwarza się w normalnym kierunku
    • reverse - animacja odtwarza się w odwrotnym kierunku
    • alternate - animacja odtwarza się w normalnym kierunku, a następnie w odwrotnym
    • alternate-reverse - animacja odtwarza się w odwrotnym kierunku, a następnie w normalnym
  7. animation - skrót dla wszystkich właściwości animacji
.element1 {
    animation-name: nazwa-animacji;
    animation-duration: 2s;
    animation-timing-function: ease-in-out;
    animation-delay: 1s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
}
.element2 {
    animation: nazwa-animacji 2s ease-in-out 1s infinite alternate;
}

Jak użyć animacji?

@keyframes zmianakoloru {
    from {
    background-color: red;
    }
    to {
    background-color: yellow;
    }
}

div {
    width: 100px;
    height: 100px;
    background-color: red;
    animation-name: zmianakoloru;
    animation-duration: 2s;
}
