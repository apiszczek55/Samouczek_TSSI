Czym jest CSS?

CSS (Cascading Style Sheets) jest używany do opisu wyglądu i formatowania strony internetowej. CSS pozwala na oddzielne zarządzanie strukturą dokumentu HTML i jego wyglądem. Można w nim kontrolować czcionkę, kolor, marginesy i wiele innych.

Walidator CSS

Walidator CSS to narzędzie, które służy do sprawdzania poprawności kodu CSS. Zarówno CSS, jak i HTML powinny być poprawnie zwalidowane, aby zapewnić zgodność z standardami.

Dołączanie arkusza stylów do dokumentu HTML

Istnieją różne sposoby dołączenia arkusza stylów do dokumentu HTML:

Dołączenie stylów bezpośrednio do elementów (styl lokalny) - poprzez użycie atrybutu style

<div style="color: white; background-color: black;">Tekst</div>

Dołączenie wewnętrznego arkusza stylów (styl wewnętrzny) - poprzez użycie znacznika <style> w sekcji <head>

<style>
    body {
        background-color: lightgray;
    }
</style>

Dołączenie zewnętrznego arkusza stylów (styl zewnętrzny) - poprzez użycie znacznika <link> z atrybutami rel="stylesheet" i href w sekcji <head>

<link rel="stylesheet" href="styl.css">

Kaskadowość

CSS jest kaskadowym językiem stylów, co oznacza, że style mogą być stosowane w różnych poziomach i mają różne priorytety. Wiele arkuszy stylów może wpływać na wygląd jednego elementu na stronie internetowej, a także jeden styl może być dołączony do wielu plików HTML. Kaskadowość określa, które style mają pierwszeństwo, gdy wiele z nich jest stosowanych do tego samego elementu.

Hierarchia stylów

(Im styl jest bliżej formatowanego elementu, tym jest ważniejszy)

  1. Styl lokalny
  2. Rozciąganie stylu <span>
  3. Wydzielone bloki <div>
  4. Wewnętrzny arkusz stylów
  5. Zewnętrzny arkusz stylów
  6. Importowany arkusz stylów
  7. Domyślne ustawienia przeglądarki

Z czego składa się CSS?

CSS składa się z reguł stylów, które definiują wygląd elementów na stronie internetowej. Każda reguła składa się z selektora i deklaracji stylu (właściwości i wartości).

selektor {
    właściwość: wartość;        <!-- Deklaracja stylu -->
}
  1. Selektor - określa, do jakich elementów zostanie zastosowany styl
    (np. div)
  2. Właściwość - określa, jakiego rodzaju modyfikację należy wykonać
    (np. kolor tła)
  3. Wartość - określa konkretną wartość właściwości (np. fioletowy)
div {
    background-color: purple;
}
